## Aim
To test and compare how different prompting patterns respond to various prompts such as **naïve (unstructured) prompts and basic (clear and refined) prompts** across multiple scenarios and analyze the **quality, accuracy, and depth** of the generated responses.

---

## AI Tools Required
- ChatGPT  
- Internet Connection  
- Computer / Laptop  

---

## Explanation

Prompting plays an important role in generating accurate responses from AI systems. Two common prompting patterns are:

### 1. Naïve Prompt
A naïve prompt is **simple and unstructured**. It does not provide detailed instructions or context.

Example:
```
Tell me about artificial intelligence.
```

### 2. Basic Prompt
A basic prompt is **clear, detailed, and structured**, providing instructions and context to guide the AI model.

Example:
```
Explain artificial intelligence in simple terms with two real-world examples.
```

In this experiment, both prompt types are tested across multiple scenarios and the responses are evaluated.

---

## Procedure

### Step 1: Define the Two Prompt Types
- **Naïve Prompt** – Simple prompt with little context.  
- **Basic Prompt** – Structured prompt with detailed instructions.

### Step 2: Prepare Multiple Test Scenarios
Different scenarios were selected such as:

1. Creative Story Generation  
2. Answering a Factual Question  
3. Summarizing a Concept  
4. Providing Advice or Recommendation  

### Step 3: Run Experiments Using ChatGPT
For each scenario:
1. Enter the **naïve prompt** and record the output.
2. Enter the **basic prompt** and record the output.
3. Compare both outputs.

### Step 4: Evaluate Responses
Responses are evaluated based on:

- **Quality**
- **Accuracy**
- **Depth**

---

## Test Scenarios and Results

| Scenario | Naïve Prompt | Basic Prompt | Observation |
|--------|--------|--------|--------|
| Creative Story | Write a story. | Write a short creative story about a robot helping a factory improve production efficiency. | Basic prompt produced a clearer and more engaging story. |
| Factual Question | What is IoT? | Explain IoT (Internet of Things) in simple terms with two examples used in manufacturing industries. | Basic prompt provided more detailed explanation and examples. |
| Concept Summary | Summarize machine learning. | Summarize the concept of machine learning in 5 simple points suitable for engineering students. | Basic prompt generated structured and clearer points. |
| Advice | Give advice for factories. | Provide three recommendations for manufacturing companies to improve efficiency using IoT technology. | Basic prompt gave more relevant and practical suggestions. |

## Analysis

From the experiment, it is observed that:

- **Basic prompts produce more accurate and structured responses.**
- **Naïve prompts often produce shorter and less detailed outputs.**
- Basic prompts guide the AI model better by providing **clear context and instructions.**
- In some simple factual questions, naïve prompts may still work reasonably well.

---

## Summary of Findings

1. Prompt clarity significantly improves AI response quality.
2. Structured prompts help the model provide **more detailed and relevant information**.
3. Providing examples and context leads to **better depth in responses**.
4. Basic prompts are generally more reliable for academic or technical tasks.

---

## Result

The comparison between **naïve prompts and basic prompts** was successfully performed across multiple scenarios.  
The results show that **basic prompts consistently generate higher quality, more accurate, and deeper responses compared to naïve prompts.**
